# React 연습용 프로젝트

이 저장소는 React 학습을 목표로 만든 개인 연습 프로젝트입니다.  
현재는 HTML 기반의 기본 구조로 시작하며,  
추후 React 및 Vite 환경으로 확장할 예정입니다.

---

## 📌 프로젝트 소개

- 프론트엔드 개발 및 React 학습을 위한 개인 실습 저장소
- GitHub 업로드 및 버전 관리 연습 목적
- 단계별로 HTML → React → 빌드/배포까지 확장 예정

---

## 🛠 사용 기술

- HTML5
- Git / GitHub

> 추후 추가 예정  
> - JavaScript (ES6+)  
> - React  
> - Vite  

---

## ▶ 실행 방법

1. 이 저장소를 컴퓨터로 내려받습니다.
2. 파일 목록에서 `index.html`을 더블 클릭합니다.
3. 웹 브라우저에서 화면이 열리면 정상 실행입니다.

---

## 📂 파일 구성

- `index.html` : 기본 HTML 페이지
- `vanilla.html` : HTML 테스트 파일
- `.gitignore` : GitHub에 올리지 않을 파일 목록
